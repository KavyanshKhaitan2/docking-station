# Ordering PCBs
These gerber zip files were produced using JLCPCB Toolkit in KiCAD, so it is recommended to order the PCBs from JLCPCB.
- All green
- All 2 layer boards
- All 1.6mm thick

# Printing the 3D models
- The Main Case, which is split in 2 parts, is printed in all black.
- - You may need to use some supports to print it properly, in which case, please do!
- The ePaper case contains two sub-modules
- - The Display angle-r (to connect the ePaper case with the Main Case at an angle)
- - - This part can be printed sideways so its efficient.
- - The ePaper case (the actual case)
- - Both have to be printed in all black.
- Switches Case
- - The "Part 1" and "Part 2" parts are printed in all-black.
- - The "Screw" part has to be printed 4 times in all-black.
- - The "Accent" part has to be printed in the accent colored filament, which should ideally be closest to this hex color: #8B2B2B
